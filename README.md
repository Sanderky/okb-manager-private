## Description
This is a comprehensive web tool designed to streamline key operations in small construction companies. Built using a modern React/TypeScript stack and utilizes Firebase as its serverless backend.

### Key features:
- File Storage / Document Management
- Employees Database
- Constructions Database
- Employee Leave Calendar (PTO Tracker)
- Work Schedule Management
- Time Tracking / Timesheet Logging

![Preview](./preview/preview.png)

## Technologies
- React 19
- TypeScript 5.8.3
- Firebase 12

## Setup

1) `npm install` 
2) Configure .env

Rename the `.env.example` file to `.env` and populate it with the actual configuration keys obtained from your Firebase Console.

3) `npm run dev`
(The application will be available at http://localhost:5173)
