## Description

This is a comprehensive web tool designed to streamline key operations in small construction companies. Built using a React/TypeScript stack and utilizes Supabase as its backend.

### Key features:

- File Storage / Document Management
- Employees Database
- Constructions Database
- Employee Leave Calendar (PTO Tracker)
- Work Schedule Management
- Time Tracking / Timesheet Logging
- Accommodation Manager (Lodging Management)

## Technologies

- React 19
- TypeScript 5.8.3
- Supabase
